2. Review each file independently against control, not against the desired conclusion.
3. List P0/P1/P2 findings with file/section anchors.
4. Separate conflicts, missing evidence, legal/professional-review gates, and optional improvements.
5. Do not silently edit files or claim approval.
6. Return a proposed patch only if Matt explicitly asks for one.
