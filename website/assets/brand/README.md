# Uncost brand assets — approved tranche 1

**Status:** repository intake only; not a website deployment or a complete/final brand package.

This directory contains the first founder-authorized public GitHub tranche from the 2026-07-16 Uncost design handoff:

- four full logo/mark masters in `logos/`;
- the fifteen canonical sector illustrations in `sectors/`;
- exact source/output hashes and metadata in `ASSET_MANIFEST.json`, `ASSET_MANIFEST.csv`, and `SHA256SUMS.txt`.

## Rights and licence boundary

## Use rules

- Preserve logo/mark aspect ratio. Never stretch, squash or crop a lockup.
- `*-light.png` contains dark/ink art for use on light/cream surfaces.
- `*-dark.png` contains light/cream art for use on dark surfaces.
- Use the informative alt text recorded in the manifest; the logo is not decorative.
- Preserve each sector illustration’s full 1531×1029 canvas and floor rule. Do not crop or reorder the sector index alphabetically; product order is the canonical fifteen-sector order.
- Repository presence does not select an asset for production, deploy it, publish a website, or clear any held asset.

## Deliberately excluded

See `HELD.md`. This tranche excludes every robot original/derivative, both clipped tight-lockup files, fonts, the icon sprite, archived sector art, pasted/generation uploads, and watermarked media.

The robot class remains held until a pack-wide transparency repair and independent visual re-audit pass. A future corrected tranche must use new hashes and explicit lineage; it must not overwrite or silently relabel a held file.
