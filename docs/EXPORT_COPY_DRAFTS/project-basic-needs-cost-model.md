## Overview

> DESIGN-AUTHORED (no source)

Where the honest chain gets tested.

> DOSSIER-BACKED: Public summary + "The problem and the cost being reduced" + "How it works"

The Basic Needs Cost Model is the underlying math: it models how AI, robotics, renewable energy, open-source hardware, local production, and shared infrastructure could reduce costs across sectors — and, crucially, where they could not.

⚠ DIVERGES FROM JULY SOURCE: this sentence expands the mechanism list to "renewable energy, open-source hardware, local production" — none of these appear in the dossier's own framing ("AI, robotics, and shared infrastructure") nor in the controlling source's list ("AI, robotics, open-source tools, and shared infrastructure"). Export-authored expansion — source wins.

A scenario only counts as a reduction path where energy, capital, maintenance, and local conditions make a net reduction plausible once everything is counted. Where the model shows automation does not beat the status quo, that negative result is published too.

The Model connects current costs to specific lower-cost paths and to fundable project budgets, so a promising idea can move from a scenario to an honestly-scoped project brief.
