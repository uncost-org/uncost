# pledge-verify

- route note: Pledge flow state
- export page: project/site/pledge-verify.html
- dossier: none
- purpose: The landing page a reader hits after clicking the confirmation link in their pledge email. It renders one of three states — the signature is confirmed, the link has expired, or the link was already used — and points the reader to the next step.
