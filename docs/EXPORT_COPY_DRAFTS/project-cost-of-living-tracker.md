## Overview

> DOSSIER-BACKED: Snapshot ("the data layer of The Case")

The data layer everything else stands on.

> DOSSIER-BACKED: Snapshot / Role in the system (supplies data to PRJ-001) + MOVEMENT_PLAN flagship (Human Essentials Dashboard)

The Tracker follows familiar prices — rent, food, energy, wages, and selected basics — with a visible source and date on every series. It is the data layer feeding the Human Essentials Dashboard and the public content, so what people see is always traceable.

> DOSSIER-BACKED: MVP deliverable 3 (staleness flags) + How it works (official-statistics-first) + Resource posture (automation a funding priority)

Because data goes stale, the Tracker is designed from the beginning to pull from public sources where possible, show "last updated" dates, and follow a regular review cadence. Automating those pipelines is a funding priority — precisely so the evidence stays current.

> DOSSIER-BACKED: Success metrics (every figure traceable to a named, dated, licensed SRC-) + MOVEMENT_PLAN trust posture ("receipts first, open corrections")

Like everything in The Case, it is receipts-first: no series is published without a named, dated, licensed source, and corrections are logged in the open.
