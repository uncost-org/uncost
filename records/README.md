# Public records

Uncost uses stable prefixes so records can be cited and changed transparently:

- `SEC-###` — sectors
- `PRJ-###` — projects
- `POL-###` — policies
- `EXP-###` — experiments
- `ASM-###` — Assembly proposals/outcomes
- `SRC-###` — sources
- `CHG-###` — material changes/corrections

No empty prefix creates authority. Records become active only through the process applicable to their type.
